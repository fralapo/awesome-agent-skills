# Cheatsheet — Fast Judgment Reference

Decision rules, compact tables, and quick heuristics. Every line helps DECIDE something.

---

## Decision Rules by Role

### Copywriter
- If headline has no benefit AND no brand name → rewrite before anything else.
- If brief has more than one objective → cut until one remains.
- If idea only works with spectacular execution → the idea is weak; find the idea.
- If someone says "the client won't buy it" before showing the work → correct them. AEs sell creative; they don't vet it first.
- If a working line is proposed for change → refuse unless the change improves the music, not just the meaning.
- If the campaign is being killed because client/CD is tired of it → fight it. Clients tire before consumers do. Run until research says stop.
- If celebrity is on the table → propose an expert instead. Expert testimonials with functional authority outperform celebrities on preference shift.
- If music is being used to solve a creative problem → stop. Music is a crutch. Find the idea first.
- If recall is high but brand preference is flat → the work failed. Track the right metric.

### Idea Generator
- If an idea requires explanation → it isn't finished. Return to brief.
- If Recognition/Surprise score matches the obvious solution → kill it.
- If the idea has only one element → find the second thing. Bisociation is non-negotiable.
- If the norm frame is mixed → fix it. Social + market signals cancel each other.
- If FREE is used without designing the social-norm covenant → redesign the whole relationship.
- If product tiers are binary (two options) → add the decoy. Never two options.
- If the brief is rational in an emotional category → question the brief, not the executions.

### Creative Leader
- If a report can deliver at 80%+ quality → coach and hand off. Step in only for irreversible stakes.
- If feedback is directional and project is at 30% → give broad notes only; never nit-pick.
- If you're doing the work your team should do → you're stealing their power. Stop.
- If the same behavior pattern appears in multiple people → the culture is producing it; diagnose tribal stage.
- If tribal stage is wrong → apply stage-specific leverage (see table below), not generic motivation.
- If "would I rehire them?" answer is no → the move is overdue.
- If conversation becomes crucial (high stakes + differing opinions + emotions) → Start with Heart before speaking.
- If someone goes silent or hostile → AMPP, not more content.
- If goals seem incompatible → CRIB before declaring deadlock.

### Visionary
- If brief can't fit in one emotionally resonant sentence → sharpen it; don't brief the team yet.
- If nothing produces the Ecstatic response → work is not there yet; return to Experimentation.
- If you're refining past the point of extraction → Demo-itis. Ship.
- If a reference is so well-executed you feel you shouldn't touch it → absorb as spark, not blueprint.
- If you're repeating what sold last season → stop. The question is never "what worked before?"
- If the team is comfortable → the brief is probably too safe.
- If a technical warning from the team arrives → taste does not override physics. Hold both.
- If you find yourself explaining the work → cut or change until you don't.

## Recognition/Surprise Matrix

| Recognition | Surprise | Verdict | Action |
|---|---|---|---|
| Low | Low | Failure | Scrap. Start over. |
| High | Low | Obvious | Too safe. Push surprise axis. |
| Low | High | Baffling | Alienating. Strengthen recognition. |
| **High** | **High** | **Hit** | **Pursue.** |

Familiarity without surprise = wallpaper. Surprise without familiarity = noise. Both axes must score high.

## Five Tribal Stages

| Stage | Language Tell | Structure | Upgrade Move |
|---|---|---|---|
| 1 — "Life sucks" | Hostility, alienation | Gang clusters | Emphasize choice. Take them where action is. Cut Stage 1 peers. |
| 2 — "My life sucks" | "We tried that," victim talk | Isolated individuals | Dyadic relationships one at a time. Short winnable projects. No team-building speeches. |
| 3 — "I'm great (you're not)" | "I don't have enough time," hoarding | Hub-and-spoke dyads | Projects bigger than one person. Tell your own Stage 3→4 story. Introduce triads. |
| 4 — "We're great (they're not)" | "We," values talk | Triadic networks | Codify values, define noble cause, Oil Changes quarterly, name an adversary. |
| 5 — "Life is great" | Wonderment, history-making | Open network | Sustain Stage 4 as the base; Stage 5 is a burst, not a destination. |

Diagnostic: listen 5 min. Count I vs. we. Do problems route to you or between peers?

## Ogilvy's Big Idea Test (5 Questions)

Run before committing any campaign. All five must be YES.

1. Did it make me gasp when I first saw it?
2. Do I wish I had thought of it myself?
3. Is it unique?
4. Does it fit the strategy to perfection?
5. Could it be used for 30 years?

Fewer than 5 yes → not a Big Idea. Return to ideation.

## Headline Approval Checklist

Before approving any headline:

- [ ] Specific benefit promised?
- [ ] Brand name included?
- [ ] Delivers news? (*now*, *introducing*, *amazing*, *suddenly*)
- [ ] Specific number or fact (not vague superlative)?
- [ ] Flags the right audience?
- [ ] Quotes around it? (+28% recall)
- [ ] No period at the end?

Fail any → rewrite.

## STATE / AMPP / CRIB — When to Use Each

| Tool | Use when… | First move |
|---|---|---|
| **STATE** | Raising a sensitive issue, confronting underperformance, defending creative direction | Share observable facts (not conclusions) |
| **AMPP** | Someone has gone cold, shut down, or escalated to hostility | Ask: "What's going on? I'd like to hear your view." |
| **CRIB** | Creative review has become a position battle; purposes seem incompatible | Commit publicly to finding mutual purpose |

Safety test before any of them: Is Mutual Purpose present? Is Mutual Respect present? If either is absent, restore it before re-entering the content.

## Fire Triangle Diagnostic

| Symptom | Weak Leg | Fix |
|---|---|---|
| Flat ideas, low energy, blank pages | **Oxygen** (margin, freedom) | Create breathing room. Reduce commitments. Protect unstructured time. |
| Compliance without investment, going-through-motions | **Fuel** (purpose, trust, empowerment) | Reconnect to why. Name the noble cause. Remove fear. |
| Quality declining, coasting, no critique landing | **Heat** (challenge, direction, expectations) | Raise the stakes. Run honest critique. Set scary goals. |
| Burnout + declining quality | Oxygen + Heat mismatch | Reduce Heat first; restore Oxygen; then reintroduce Heat. |

Never apply Heat to a team without Oxygen. Burnout, not performance, results.

## Four Feedback Types

| Type | When | Format |
|---|---|---|
| Expectation-setting | Before work begins | State what great looks like, what mediocre looks like, common pitfalls |
| Task-specific | Immediately after an action | Direct, with specific example |
| Behavioral | Pattern repeats across multiple tasks | Connect the dots; don't isolate each instance |
| 360-degree | Your direct message is not landing | Aggregate from multiple sources; run quarterly |

Rule: 50%+ of feedback volume should reinforce what is working. Silence reads as abandonment to top performers.

## Persuasion Levers

| Effect | Creative Use |
|---|---|
| **Anchoring** | Lead proposals and presentations with highest-value option. Everything after is judged relative to it. |
| **Decoy Effect** | Three options, never two. Make the lower option inferior to middle on one clear dimension. |
| **Endowment Effect** | Enable genuine use in trial. "Your X" language. Configurators. Personalization before payment. |
| **Zero Price Effect** | FREE triggers social-norm mode. Design the entire subsequent relationship as a social-norm exchange or don't use FREE. |
| **Social vs Market Norms** | One monetary signal destroys a warm social frame permanently. Choose the frame; never mix. |
| **Price-Placebo Effect** | Higher price = objectively better experience. Discounting degrades the product in the mind. Never discount without strategic intent. |
| **Self-Herding** | First-use rituals carry more conversion power than loyalty programs. Design on-boarding as a category-defining moment. |
| **Hot-Cold Gap** | Design for the emotional state at decision moment, not the rational at-home state. Point-of-sale is emotional. |
| **Priming** | Content before your content is part of your content. Own the sequence. |

## Tells & Smells — Fast Trouble Heuristics

**"The client won't buy it."** Said before showing the work → restricting to what's expected, not what's needed. Correct it immediately.

**"It's a bit like [famous campaign]."** Said with admiration during ideation → you're working on the obvious. The famous campaign is the brief, not the answer.

**Unanimous enthusiasm in a committee.** Committees cannot create. Most bad campaigns are the minutes of a committee. The sharper the idea, the more polarizing the room.

**"We should add music."** Creative energy is low and there's no idea yet. Music before idea = papering over absence.

**"Let's just make it simpler."** Often means "let's make it safer." If simplicity is the goal, demand it be Hemingway-simple, not brief-compliant-simple.

**"The team is working really hard."** Effort in the absence of output quality = wrong direction, not insufficient effort. Diagnose Oxygen/Fuel/Heat before praising the effort.

**Everyone nods in the meeting, no one implements.** Pool of Shared Meaning is shallow. Hard truths are being said in hallways, not rooms. Safety problem.

**New project starts with "last time we did..."** Clean Table discipline has collapsed. Stop. Reset before brief.

**Brief has three objectives.** It has no objectives. Return it with one objective or refuse to brief the team.

**"I don't like it."** Not a professional assessment. Replace with "it doesn't work because..." then name the functional failure.

**Reverse-type layout (white on black).** Proven across decades to destroy readership. No exception.

**"They're just not motivated."** Diagnostic shortcut. Is Oxygen present? Is purpose clear? Is challenge calibrated? Motivation is an output, not an input.

**"This is a career-defining project."** Fear-based framing. Creates compliance and kills the kid-like innocence that produces the best work.
