# Cheatsheet

## Cognitive Bias Catalog (Ch5)
| Bias | One-line effect |
|---|---|
| Confirmation bias | Seeks/remembers only belief-confirming info |
| Consistency bias | Sticks to past position to look consistent |
| Dichotomous thinking | Collapses a spectrum into two extremes |
| Peacock bias | Shows only successes, hides failures |
| Response bias | Negative responders skip surveys, skewing results positive |
| Projection bias | Assumes others think like you; agreement feels like proof |
| Negativity bias | Weighs negative info more than equal positive info |
| Optimism bias | Overestimates good outcomes, underestimates risk |
| Representativeness heuristic | Vivid specific detail feels more probable than general |
| Anchoring effect | An arbitrary number skews a later estimate |
| Availability heuristic | Ease of recall substitutes for actual frequency |
| Halo effect | Order of traits shapes interpretation of all that follow |
| Conformity pressure | Echoes group's wrong answer over own evidence |
| Belief/mindset effect | Belief about a stimulus changes physiological response |
| Stress-enhanced memory | Moderate arousal strengthens memory of that moment |
| Brain drain (attention residue) | Visible phone reduces cognitive capacity |
| Situational power (Lucifer effect) | Role/context can override personal character |
| False memory / Mandela Effect | Memory is reconstructed, not recorded — distortable |

## Logical Fallacy Catalog (Ch4)
| Fallacy | Move | Counter |
|---|---|---|
| Strawman | Attacks an exaggerated version of the claim | Ask for the exact link to the original claim |
| Ad hominem | Attacks the person, not the argument | Name it, restate the point |
| Poisoning the well | Discredits the person before they speak | Name the setup explicitly |
| Tu quoque | "You do it too" deflection | Point out it's irrelevant to the claim |
| Appeal to ignorance | Unproven = false (or vice versa) | Absence of proof ≠ proof of absence |
| Ad populum | True because many believe it | Popularity isn't evidence |
| Ipse dixit (authority) | True because an "expert" says so | Ask what the evidence says |
| Anecdotal fallacy | One vivid case overrides the data | Ask for the base rate |
| False analogy | Surface similarity treated as equivalence | Test if the similarity is causally relevant |
| False dichotomy | Only two options presented | Name a third option |
| Correlation-causation | Two correlated things assumed causal | Ask what third factor explains both |
| Post hoc | A before B assumed A caused B | Ask for the mechanism |
| Cherry picking | Only supporting evidence shown | Ask what was excluded |
| Etymological fallacy | Original word meaning must still apply | Language evolves — ask current meaning |
| Appeal to nature | Natural = good, artificial = bad | Counterexamples break it instantly |
| Fallacy fallacy | Bad argument ⇒ false conclusion | Bad argument only proves itself weak |

## Speech-Opening Techniques (Ch7, Ch3)
| Technique | Use when |
|---|---|
| Question opening | Audience already lives the problem |
| Shocking factoid | You have a true, verifiable, surprising stat |
| Story opening (strongest) | Any talk over a few minutes |
| 60-second hook | Short-form / elevator context |
| Never: self-introduction | — signals disengagement, avoid always |

## 4 Types of Audience (Ch9)
| Type | Opening move |
|---|---|
| Hostile | Reopen-the-Decision: affirm past choice, add new info, ask to reconsider |
| Friendly | Unification — deepen connection, not persuade |
| Apathetic | Lead with relevance ("what's in it for you") |
| Uninformed | Give evaluation criteria before the conclusion |

## Anxiety Ladder (Ch8, weakest → strongest)
1. Remove the source (avoid) — no growth
2. Manage the level (breathing, self-talk)
3. Tolerate it (coexist without behavior change)
4. Enjoy it (connect to purpose bigger than self)

## Close Selection (Ch13)
| Situation | Close |
|---|---|
| Low-friction ask, audience leaning your way | Direct — name the action |
| Bigger ask, audience needs to feel the cost of inaction | Indirect — dramatize the cost |

## 3 Pillars / 7 Elements / 5 Voice Aspects — quick counts
- **3 Pillars of Persuasion**: Logos, Ethos, Pathos
- **7 Elements of a Pitch**: Problem, Who, Product, Competition, Make Money, Resources, Capture Value
- **5 Aspects of Voice**: Breathing, Resonance, Silence, Articulation, Inflection
- **4 Levels of Question**: Fact → Opinion → Impact → Change
- **5 Paths to Communication Failure**: unclear want, unclear plan, no resources/execution, no self-belief, no rehearsal
